A clean and responsive "Join Our Newsletter" landing page with a centered card layout, modern typography and mobile-first approach.

### What I learned

- How to create responsive layouts using media queries
- Centering content both vertically and horizontally in different screen sizes
- Working with Google Fonts (Poppins) and proper font loading
- Creating consistent spacing and typography scale across devices
- Styling form elements (input + button) with focus states and placeholder customization
- Using box-shadow and border-radius to create modern card design
- Mobile-first CSS approach with progressive enhancement for larger screens

### Useful resources

- [Google Fonts - Poppins](https://fonts.google.com/specimen/Poppins) - Great modern sans-serif font family
- [devChallenges - Join Our Newsletter](https://devchallenges.io/challenge/join-our-newsletter) - Original challenge page with design
- [CSS Flexbox Guide](https://css-tricks.com/snippets/css/a-guide-to-flexbox/) - Helpful Flexbox reference

## Built with

- Semantic HTML5 markup
- CSS custom properties (not used in this version, but could be added)
- Flexbox
- Mobile-first responsive design
- Media Queries
- Google Fonts (Poppins)

## Features

- Responsive design (mobile, tablet, desktop)
- Centered layout with logo, headline, description and subscription form
- Clean modern card with subtle shadow
- Styled email input and prominent subscribe button
- Confidentiality message below the form
- Hover/focus states on interactive elements (can be improved)
- Author credits with links at the bottom

This application/site was created as a submission to a [DevChallenges](https://devchallenges.io/challenges-dashboard) challenge.
